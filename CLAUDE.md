## Commands

```bash
bun run dev        # Start dev server on port 3003
bun run build      # Production build
bun run test       # Run tests with vitest
bun run lint       # ESLint
bun run typecheck  # TypeScript type checking (tsc --noEmit)
bun run format     # Prettier (write)
bun run check      # Prettier (check only)
```

Run a single test file: `bun run test -- path/to/file.test.tsx`

## Architecture

This is a **TanStack Start** (SSR-capable React meta-framework) app using **file-based routing** via `@tanstack/react-router`.

**Routing**: Routes live in `src/routes/`. The router is configured in `src/router.tsx` and `src/routes/__root.tsx` defines the shell. `src/routeTree.gen.ts` is auto-generated by the router plugin — never edit it manually.

**UI components**: shadcn/ui components are added via `npx shadcn add <component>` and land in `src/components/ui/`. The config is `components.json` (style: `radix-rhea`, base color: olive, icon library: tabler).

**Styling**: Tailwind CSS v4 via `@tailwindcss/vite`. Theme tokens are CSS custom properties defined in `src/styles.css` using OKLCH colors. Dark mode uses the `.dark` class. The `@/*` path alias maps to `src/`.

**Icons**: Use `@tabler/icons-react` (configured as the shadcn icon library).

**Testing**: Vitest + `@testing-library/react` with jsdom environment.
